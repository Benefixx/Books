### Это панель администратора. Тут администратору показываются, какие книги у кого находятся, на данном скрине только 1 книга находится у пользователя, тут у нас доступны 3 функции, первая функция позволяет удалить книгу из базы если она утеряна и т.д, вторая функция "Попросить вернуть книгу" после её нажатия пользователю который взял эту книгу ему в телеграм придет сообщение от бота автоматически с просьбой вернуть книгу, бот пишет человек название книги и дату когда он брал эту книгу, и 3 функция "подтвердить получение книги", когда пользователь возвращает книгу в библиотеку, работнику надо нажать на эту кнопку, чтобы её опять выложить на сайт"
